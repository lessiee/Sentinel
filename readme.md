✨ Overview

Sentinel is a high-performance proxy intelligence engine designed to inspect, verify, benchmark, and classify proxy servers using an adaptive asynchronous architecture.

Unlike traditional proxy checkers that simply determine whether a proxy is alive or dead, Sentinel performs an extensive inspection of each endpoint to build a detailed intelligence profile.

The engine automatically adjusts its own concurrency, analyzes network behavior, performs reliability testing, evaluates anonymity, calculates quality scores, detects provider information, and generates professional reports suitable for large-scale proxy validation workflows.

The goal is simple:

> Transform raw proxy lists into actionable intelligence.




---

🌍 Philosophy

Sentinel was built around one principle.

A proxy should never be judged only by whether it responds.

Instead, every proxy should answer questions such as:

🔹 Where is it located?

🔹 Who owns it?

🔹 Is it residential?

🔹 Is it anonymous?

🔹 Can it reliably serve requests?

🔹 How stable is the connection?

🔹 Is it blacklisted?

🔹 How trustworthy is it?

Only after collecting these answers does Sentinel generate an overall quality score.
